================================================================================
                           HOW TO TEST
================================================================================

1. Open URL Detection page
2. Select "Model (ML-based)" mode
3. Enter test URL from above
4. Click "Check URL"
5. Verify result matches expected category

For batch testing:
- Switch to "Batch URL Analysis" tab
- Paste URLs (one per line) or upload file
- Click "Analyze All URLs"
